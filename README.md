# Doofus Game

A 3D platformer where players jump across dynamically spawning platforms that disappear over time.

## Gameplay

[![Gameplay Video](https://img.youtube.com/vi/yRpYu3oWKUs/0.jpg)](https://youtu.be/yRpYu3oWKUs)

## Features

- Force-based physics movement with levitation
- Dynamic platform spawning and destruction
- Score tracking system
- Visual warning indicators

## Controls

- WASD / Arrow Keys: Move
- Mouse: UI navigation

## Setup

1. Clone repository
2. Open in Unity 2021.3+
3. Load main scene
4. Press Play
